# 리액트 훅스

- 원래 17년도 이전 리액트에선 class 컴포넌트가 주류였음
- class 컴포넌트에서 useState, Ref 등등 좋은 기능들을 사용할수 있었고 반면 function 컴포넌트는 ui 리랜더링정도의 기능밖에 없었음
- 다만 class의 문법이 너무 복잡해 function을 컴포넌트로하며 class의 기능들을 사용할 수 있는것에 대한 수요가 많아졌고 이를바탕으로 React Hooks 라는 기능이 생김
- 리액트 훅스는 function 컴포넌트에서 class 컴포넌트에서 사용하던 기능들을 사용할수 있게 하는것임
- 그래서 배웠던 useState, useRef 모두 리액트 훅스를 통해서 사용했던것임

- 리액트 훅스에는 use라는 접두사가 붙음.
- 특징
  - 리액트 훅스는 함수 컴포넌트에서만 사용 가능
  - 조건문, 반복문 내에선 사용 불가
  - 나만의 커스텀 Hook도 제작 가능
