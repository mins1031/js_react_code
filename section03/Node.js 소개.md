Node.js
Node.js는 브라우저가 아니더라고 js 를 실행할 수 있게 해주는
JavaScript 실행환경(=Run Time) 이다. = js 구동기

- ex) 닌텐도 = Node.js, 닌텐도 칩 = JavaScript
- ex) JVM = Node.js, JAVA = JavaScript
