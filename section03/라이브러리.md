Node.js 라이브러리

라이브러란?

- 프로그램을 개발할때 필요한 다양한 기능들을 미리 만들어 모듈화 해놓은 것.
- ex) 날짜, 수학, 그래픽 라이브러리

스프링 쓰면서 빌드툴에 라이브러리 의존성 넣어놓듯이 같은 개념인듯?

randomcolor라는 라이브러리를 설치하면
package.json에 아래와 같은 항목이 추가됨.
"dependencies": {
"randomcolor": "^0.6.2"
}

참고로 node_modules 와 package-lock.json 파일이 라이브러리 설정파일 + 코드 보관 파일인데 이게 없으면 원칙적으론 라이브러리 임포트가 실패함.
그래서 npm run 시킬때 해당 파일들에 없는 내용은 자동으로 포함하게 하는 명령문이 존재

명령어 : npm i
-> package.json의 dependencies에 설정된 라이브러리를 다운로드 받는 명령어.
그래서 만약 node 를 누군가에게 공유해야 한다면 node_modules 라는 무거운 폴더는 삭제후 공유해도됨.
